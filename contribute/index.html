If you want to contribute to the O-FISH application, find an issue to solve:
  1. Android mobile app issues
  2. iOS mobile app issues
  3. Node.js web app issues
You will need to fork the appropriate code repository (Android, iOS, Web) and create a NEW BRANCH to code your fix. For example, to clone the web app and create a branch called 'fix_issue_12345':

git clone https://github.com/WildAid/o-fish-web
cd o-fish-web
git checkout -b to_fix_issue_12345

Code your fix and build the O-FISH application to test that it works.

Once you verify the issue is fixed, you will need to submit your patch to the github code. Commit, and push, e.g.:

git commit -m 'fixing issue 12345'
git push origin fix_issue_12345

Create a pull request as the output instructs:

remote: Create a pull request for 'fix_issue_12345' on GitHub by visiting:
remote: https://github.com/WildAid/o-fish-web/pull/new/fix_issue_12345

A developer will review your code, and once the code is ready it will be merged in.

If you have any questions, connect with us on the MongoDB Community Forum for Open Source Projects.
